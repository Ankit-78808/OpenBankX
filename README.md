# OpenBankX
 
### Open Banking / PSD2 API Gateway & Consent Management System
 
OpenBankX is a simplified Open Banking platform inspired by PSD2 standards.  
It is designed to demonstrate secure authentication, Third Party Provider (TPP) onboarding, consent management, and controlled data access through a modular banking API architecture.
 
The project aims to help developers understand the architecture behind modern open banking systems and financial API ecosystems through a practical implementation.
 
---
 
## Features
 
- Secure authentication and authorization
- TPP (Third Party Provider) onboarding
- Consent management system
- Secure API gateway architecture
- Microservice-friendly design for financial systems
 
---
 
## Tech Stack
 
- Java
- Spring Boot
- Maven
- MySQL
- REST Architecture
- Git & GitHub
 
---
 
## Project Structure
 
```
src/
 ├── controller/
 ├── service/
 ├── repository/
 ├── model/
 └── config/
```
 
This structure follows a layered architecture separating controllers, services, repositories, and configuration files to maintain clean and scalable code.

## Setup Instructions
 
### 1. Clone the repository (master branch)
 
```
git clone -b master https://github.com/Ankit-78808/OpenBankX.git
```
 
### 2. Navigate to the project folder
 
```
cd OpenBankX
```
 
### 3. Run the application
 
```
mvn spring-boot:run
```

# 🚀 **OpenBankX**
### **Open Banking / PSD2 API Gateway & Consent Management System**

OpenBankX is a simplified Open Banking platform inspired by PSD2 standards.  
It provides secure authentication, TPP onboarding, consent-based data access, and a controlled API gateway — ideal for learning real-world financial API workflows.

---

## **🛠️ Tech Stack**

### **Frontend**
- Angular

### **Backend**
- Java Spring Boot

### **Database**
- MySQL

---

## **🔐 Core Features**
- **OAuth2/OIDC authentication** with PKCE  
- **Strong Customer Authentication (SCA)** using OTP  
- **TPP onboarding** and application registration  
- **Consent management** (create, view, revoke)  
- **API Gateway** with token + consent validation  
- **Mock banking APIs** (Accounts, Balances, Payments, Transactions)  
- **Monitoring & analytics** (usage, errors, latency)  
- **Audit & compliance logs**  
- **Notifications & alerts** (consent expiry, rate-limit events)

---

## **🧩 Modules Included**
- **Identity, OAuth2, OIDC & SCA**  
- **TPP Onboarding & App Registration**  
- **Consent & Grant Management**  
- **API Products & Rate-Limiting**  
- **API Gateway & Routing**  
- **Accounts & Payments Reference APIs**  
- **Monitoring & Analytics**  
- **Audit & Reporting**  
- **Notifications & Alerts**

---

## **🎯 Purpose**
OpenBankX helps developers understand open banking architecture, PSD2 flows, secure API development, and scalable backend design through a practical, hands-on implementation.
