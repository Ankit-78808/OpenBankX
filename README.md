🚀 OpenBankX
Open Banking / PSD2 API Gateway & Consent Management System
OpenBankX is a simplified, developer‑friendly implementation of Open Banking concepts.
It includes secure authentication, TPP onboarding, consent management, and a controlled API gateway—built for learning PSD2 workflows, microservices, and financial‑grade API design.

🛠️ Tech Stack
Frontend

Angular

Backend

Java Spring Boot

Database

MySQL


🔐 Core Features

OAuth2/OIDC authentication with PKCE
Strong Customer Authentication (OTP-based)
TPP onboarding & application registration
Consent management (create, view, revoke)
API Gateway with token + consent validation
Mock banking APIs (Accounts, Balances, Transactions, Payments)
Monitoring & analytics (usage, errors, latency)
Audit & compliance logs
Notification system for consent expiry, alerts, etc.


🏗️ Modules

Identity, OAuth2, OIDC & SCA
TPP Onboarding
Consent & Grant Management
API Products, Plans & Throttling
API Gateway
Accounts & Payments APIs
Monitoring & Reporting
Notifications & Alerts


🎯 Purpose
This project showcases how Open Banking platforms work internally — ideal for learning PSD2, secure API development, and enterprise‑grade backend architecture.
